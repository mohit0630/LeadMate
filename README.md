# LeadMate 🤖

> Never miss a customer again.

AI-powered WhatsApp agent for Indian small businesses. Handles bookings, FAQs, and customer queries 24/7 in your brand's voice.

## Tech Stack
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Groq API (llama-3.3-70b-versatile)

## Setup

1. Clone the repo
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env.local`
4. Add your Groq API key (free at https://console.groq.com/keys)
5. Run: `npm run dev`

## Deploy on Vercel

1. Push to GitHub
2. Import repo on Vercel
3. Add `GROQ_API_KEY` in Vercel environment variables
4. Deploy

## Features
- Live AI demo (4 business types: Salon, Clinic, Restaurant, Coaching)
- WhatsApp-style chat UI
- Landing page with pricing
- Lead capture form
- Fully mobile responsive
